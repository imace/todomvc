# TodoMVC
TodoMVC implementation for Widok.

## Installation
Compile Widok:

```
git clone git@github.com:widok/widok.git
cd widok
sbt publish-local
cd ..
```

Compile TodoMVC:
```
git clone git@github.com:widok/todomvc.git
cd todomvc
sbt fastOptJS
```

Now open ``application.html`` in the browser.
